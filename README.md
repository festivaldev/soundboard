# FSVL Soundboard

## Build instructions
```bash
# Install dependencies
npm install

# Build Vue app before Electron
npm run build

# Build for Windows (32 bit)
npm run build-win
# Build for Windows (64 bit)
npm run build-win64
# Build for macOS (Intel)
npm run build-osx
# Build for macOS (Apple Silicon)
npm run build-osx-arm64
```
